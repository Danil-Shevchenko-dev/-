Simple Calculator (Python CLI)

A simple command-line calculator written in Python. It performs basic arithmetic operations and runs in a loop until the user exits.

Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Continuous operation in a loop
* Exit option by entering 0 as the first number

How It Works

The program repeatedly:

1. Asks for the first number
2. Lets the user choose an operator
3. Asks for the second number
4. Prints the result
5. Repeats until the user enters 0
